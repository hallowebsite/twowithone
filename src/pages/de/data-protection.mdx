# Datenschutz

Wir sammeln keine Daten über Besucher dieser Seite. 

## Cookies und ähnliche Technologien

Wir verwenden kein Analysesystem, keine Marketing- oder Werbelösungen, so dass wir keine
speichern keine Daten auf Ihrem Gerät.

Wenn Sie das Thema von hell auf dunkel oder umgekehrt ändern, werden wir
Informationen über Ihre Wahl auf Ihrem Gerät im so genannten lokalen Speicher Ihres Browsers speichern.
Browser, aber diese Informationen werden nicht an unseren Server gesendet.

**Das ist alles, genießen Sie Ihren Besuch.**
